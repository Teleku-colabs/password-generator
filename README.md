# 🔐 Secure Password Generator

A clean, secure, and professional password generator built with Streamlit.

**Live Demo:** (will appear after you deploy)

## ✨ Features
- Always includes **1 uppercase + 1 lowercase + 1 number + 1 symbol**
- Uses Python `secrets` modules (cryptographically secure)
- Simple and beautiful interface
- One-click copy to clipboard

## 🚀 How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
